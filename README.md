# Object Detection and Tracking using YOLOv8

This project detects and tracks objects in a video using Python, OpenCV, and YOLOv8.

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics

## Features
- Detects objects in video
- Shows object labels and bounding boxes
- Tracks objects frame by frame
- Press `q` to stop the program

## How to Run
1. Install required libraries:
   pip install ultralytics opencv-python

2. Place a video file named `sample.mp4` in the project folder.

3. Run the project:
   python main.py

## Note
I used a sample video because my laptop camera was not working. The system can also work with a webcam by changing the video input to camera input.
